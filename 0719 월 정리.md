### 변수

* 변수는 할당 연산자 = 를 통해 값을 할당
* type() 변수에 할당된 값의 타입 반환
* id() 변수에 할당된 고유한 id 값이며, 메모리주소

```python
x = 'ssafy'

type(x)
#str
    
idx(x)
#4645387184
```



### 할당 연산자

* 같은 값 동시에 할당 가능
* 다른 값 동시에 할당 가능

```python
x = y = 1004
print(x,y)
#1004 , 1004

x, y = 1, 2
print(x,y)
#1 2
```
